# java-api-boilerplate

**Version:** 0.0.1-SNAPSHOT  
**Group ID:** br.com.wendelnogueira

## Description

This project is a boilerplate API built with Spring Boot and Java. It includes essential dependencies and configurations to kickstart your development.

## Features

- Spring Boot Actuator for monitoring.
- Spring Data JPA for database interactions.
- Spring Security for authentication and authorization.
- OpenAPI integration for API documentation.
- MySQL database support.
- Lombok for reducing boilerplate code.

## Build and Run

### Prerequisites

- Java 21
- Maven

### Build

```bash
mvn clean install
```

### Run

```bash
mvn spring-boot:run
```

## API Documentation

The API documentation is available at `/swagger-ui.html` after starting the application.
